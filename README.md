# JPlusUI 3.0 全新升级 ！

JPlusUI 是一个完整的 HTML5 UI 库。

* 无需设计：快速搭建漂亮页面；同时支持电脑和手机！

* 300+ 个组件：复制示例即可使用，快速上手。

* 更懂国情：适合中文排版，提供城市组件。

* 轻量、高效：完全开源和免费。

#### 加入我们

jPlusUI 目前只在内部使用，尚未对外发布。如果您支持本库，可先加入 QQ 群：177503954 。

#### 协议

本库完全免费和开源，不作任何限制。但不得基于本库开发新的 UI 库然后作为产品盈利。
